## 👋 Hakkımda

Kapadokya Üniversitesi Bilişim Güvenliği Teknolojisi mezunu (GNO 3.28/4.00), siber güvenlik, ağ savunması ve güvenli yazılım geliştirme alanlarında akademik ve pratik eğitim almış bir sistem uzmanıyım. Donanım seviyesinden işletim sistemlerine, ağ yönetiminden temel güvenlik pratiklerine kadar geniş bir yelpazede BT süreçlerini yönetiyor; Python ile süreçleri otomatikleştiriyor, sistem sorunlarını kökünden çözüyorum.
